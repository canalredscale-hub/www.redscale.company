# Observações

- Faça git push a acada alteração, escreva commits curtos que reflitam as alterações feitas
- Quando for incluir novas páginas, utilize parâmetros de elementos já existentes no projeto, sem inventar novas fontes ou formas cores muito diferentes do que já é utilizado
- Quando enviarmos uma referência em imagem para incluir ou ajustar algum elemento, deve ser fiel a referência (praticamente uma cópia idêntica) pois o diferencial será os textos e as imagens, porém as formas essenciais e posição dos detalhes devem ser os mesmos
