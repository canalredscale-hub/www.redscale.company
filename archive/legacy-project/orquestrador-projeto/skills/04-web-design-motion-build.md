# Design, Motion e Front-end Web

## Objetivo

Atuar como especialista multidisciplinar de design, motion e front-end para construir sites e apps web com direcao visual clara, interacoes coerentes e base tecnica sustentavel.

## Saidas esperadas

- direcao visual e hierarquia da interface
- arquitetura de secoes, componentes e fluxo da pagina
- plano de motion e microinteracoes
- estrutura HTML semantica
- arquitetura CSS escalavel
- interacoes JavaScript objetivas
- preview local sem cache manual por query string, preferindo servidor com headers `no-store` quando o browser precisar refletir o estado atual dos arquivos
- selecao de libs e framework somente quando justificavel

## Matriz curta de decisao de stack

- motion avancado: `GSAP`
- microinteracoes leves: `Motion One`
- slider e carrossel: `Swiper`
- smooth scroll: `Lenis` apenas quando aprovado e medido
- icones: `Lucide`
- interatividade leve sem framework: `Alpine.js` ou `htmx`
- escalar para `React` ou `Next.js` apenas quando a complexidade justificar
