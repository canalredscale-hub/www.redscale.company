# Skills

Skills locais para orientar como o agente deve executar tarefas recorrentes deste projeto com o workflow `web_app`.

## Catálogo recomendado

### `01-context-rebuild.md`

- Título: Reconstrução de contexto
- Resumo: Ler a memória essencial e resumir o estado atual do projeto.
- Pontos principais:
  - onde estamos
  - próximo passo
  - decisões pendentes

### `02-milestone-planning.md`

- Título: Planejamento de marco
- Resumo: Quebrar o próximo marco em passos curtos, com dependências, riscos e validação.
- Pontos principais:
  - plano enxuto
  - dependências explícitas
  - critérios de saída

### `03-execution-cycle.md`

- Título: Execução de ciclo
- Resumo: Executar apenas o necessário para a etapa atual e registrar o handoff correto.
- Pontos principais:
  - entrega do ciclo
  - evidências de validação
  - atualização da memória do projeto

### `04-web-design-motion-build.md`

- Título: Design, Motion e Front-end Web
- Resumo: Guiar a construção visual e técnica de sites e apps web com foco em direção visual, motion, HTML, CSS, JS e uso criterioso de bibliotecas.
- Pontos principais:
  - direção visual e hierarquia
  - motion e microinterações
  - seleção de stack e libs
