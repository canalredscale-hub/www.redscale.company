Sobre o conceito

Este novo site foi desenhado para posicionar o Redscale como produto, não só como uma página derivada da marca mãe.

A decisão visual mistura atmosfera editorial, operação e um senso de precisão que combina com software de clareza operacional.

Clareza

Textos curtos, hierarquia forte e menos elementos concorrendo pela atenção.

Ritmo

Seções amplas, com uma ideia dominante por bloco, para deixar a leitura respirar.

Presença

Tipografia, cor e imagem trabalhando juntas para dar ao Redscale uma assinatura própria.

Direção

O objetivo não foi copiar a vitrine da Webflow, e sim usar a simplicidade estrutural dela como ponto de partida para uma identidade original do Redscale.

Por isso a navegação continua curta, a arquitetura segue multipágina e a composição continua direta. O que muda é a assinatura visual, a narrativa e a função comercial.

Base técnica

A implementação foi mantida simples para acelerar a próxima rodada de edição.

HTML estático

Quatro páginas independentes, fáceis de abrir, editar, revisar ou integrar depois.

CSS único

Um único arquivo de estilos para manter consistência visual e evolução rápida.

Inspector

Overlay de nomes dos elementos com hover, foco, persistência local e toggle manual.
